MySQL Introduccion

Una base de datos como MySQL puede ser algo simple o puede ser tan complejo como un gran sistema de administración de bases de datos relacionales, completo con herramientas integradas para ayudarlo a mantener los datos.

MySQL se define como una base de datos relacionales, ya que existen base de tados relacionales y no relacionaes como lo es por ejemplo Mongo DB, en la cual los elementos de la base de datos no tienen ninguna relacion entre si.

Métodos para almacenar datos

Antes de entrar en sistemas de administración de bases de datos de MySQL, veamos algunos métodos comunes para almacenar datos.

Archivo de texto:

Imagina que tenemos un archivo de texto llamado Artists.csv, y que los contenidos se parecen a esta captura de pantalla. Este es un archivo de texto. Más específico, es un archivo con los valores separados por comas (CSV). Las comas separan cada campo dentro de una fila.

Hoja de cálculo:

Otra opción sería almacenar los datos en una hoja de cálculo utilizando el software de hoja de cálculo (por ejemplo, Microsoft Excel). De esa manera, podríamos hacer algunas cosas adicionales con nuestra lista (como formatearla u ordenar por nombre de artista, etc.).

MySQL

MySQL es un sistema mas complejo que un archivo de texto o un archivo excel, ya que en la base de datos hay una relacion entre los distintos elementos como lo puede ser por ejemplo la relacion entre una tabla de alumnos, una tabla de profesor y una tabla de salarios de los profesores
