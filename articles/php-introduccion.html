PHP introduccion

PHP es uno de los lenguajes mas usados hoy en día. El idioma tiene la capacidad de comunicarse de un lado a otro con un servidor y crear una página web dinámica para el usuario. Si tiene una cuenta de alojamiento web para ejecutar su blog personal o sitio web, puedo garantizarle que PHP ya está instalado.

Incluso la página que estás viendo en este momento fue creada por PHP. Si tiene ambiciones de convertirse en un desarrollador web, PHP es una necesidad para usted y le permitirá crear páginas web dinámicas basadas en información externa, generalmente del usuario.

PHP no es el lenguaje para saltar directamente si no ha tenido ninguna experiencia. Comprender los conceptos básicos de programación puede ser más fácil en un lenguaje como Visual Basic, ya que la sintaxis y los conceptos en PHP pueden ser bastante confusos para un principiante.

Archivo de texto:

Si no tiene experiencia en secuencias de comandos, también podría sugerir echar un vistazo a JavaScript, que es un lenguaje de secuencias de comandos del lado del cliente, lo que lo limita en comparación con PHP. Pero no dejes que eso te asuste, PHP enseña algunos buenos hábitos de codificación y si eres capaz de aprender cosas rápidamente, entonces deberías estar bien.
